Sobre o Space

Bem-vindo ao Space - Uma Galeria de Fotos Espaciais!


Este projeto é um site desenvolvido com Django que exibe imagens espaciais. A base do projeto foi criada durante minhas aulas na escola de programação Alura, porem ao fim das aulas o projeto ainda estava muito incompleto e repetitivo então decidi deixa-lo mais robusto por conta própria, tambem deixarei o projeto inicial na descrição para comparação.


Nesta galeria, você encontrará:

  • • Nebulosa Carina: Uma das maiores e mais brilhantes nebulosas visíveis no céu noturno
  • • Nebulosa Águia: Famosa pelos seus "Pilares da Criação"
  • • Nebulosa Anel: Uma fascinante nebulosa planetária
  • • Nebulosa Cabeça de Bruxa: Uma nebulosa de reflexão com forma peculiar
  • • Nebulosa Cabeça de Cavalo: Uma das nebulosas mais reconhecíveis do céu
  • • Nebulosa Helix: Conhecida como o "Olho de Deus"

Este projeto foi desenvolvido utilizando:

  • • Django - Framework web em Python
  • • HTML5 e CSS3 para estruturação e estilização
  • • Sistema de templates do Django
